Timestampable Entity in Symfony 2.0 – 2.3

March 9, 2012How to's, Symfony DevelopmentKonrad Podgórski

This is another copy & paste ready snippet for rapid application development in Symfony 2.0. This time for well know from Symfony 1.4 Timestampable behavior.

Update 12 april 2013

  • added return $this for chaining

Update: 12 march 2012

  • fixed minor typo

 

Annotate Class with:

@ORM\HasLifecycleCallbacks

and inside class

/**
 * @ORM\Column(name="created_at",type="datetime") 
 */
private $createdAt;

/** 
 * @ORM\Column(name="updated_at",type="datetime") 
 */
private $updatedAt;

public function getCreatedAt() {
    return $this->createdAt;
}

/**
 * @ORM\PrePersist()
 */
public function setCreatedAt($createdAt = null) {
    $this->createdAt = null === $createdAt ? new \DateTime() : $createdAt;
    return $this;
}

public function getUpdatedAt() {
    return $this->updatedAt;
}

/**
 * @ORM\PrePersist()
 * @ORM\PreUpdate()
 */
public function setUpdatedAt($updatedAt = null) {
    $this->updatedAt = null === $updatedAt ? new \DateTime() : $updatedAt;
    return $this;
}
