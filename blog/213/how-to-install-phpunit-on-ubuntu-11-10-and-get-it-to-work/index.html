How to install PHPUnit on Ubuntu 11.10 and get it to work

January 7, 2012How to's, Linux, Symfony DevelopmentKonrad Podgórski

Installing PHPUnit is not a big deal, but I found that it doesn’t work out of box if you install it from Ubuntu repositories. After apt-get you still need to update it to newer version

sudo apt-get install phpunit
sudo pear channel-discover pear.phpunit.de
sudo pear channel-discover components.ez.no
sudo pear channel-discover pear.symfony-project.com
sudo pear upgrade
sudo pear upgrade phpunit/PHPUnit
