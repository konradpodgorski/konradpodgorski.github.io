Installing LAMP on Ubuntu 11.10 / 11.04 / 10.10 / 10.04

January 7, 2012LinuxlampKonrad Podgórski

Abbreviation LAMP stands for Linux Apache Mysql PHP, everything (well.. almost ) you need to start developing your awesome PHP web applications

To install lamp package simply type into terminal

sudo apt-get install lamp-server^

you might also find useful phpmyadmin

sudo apt-get install phpmyadmin

Yes, that’s all
