[How to] Call to undefined method myUser::setReferer.

October 2, 2011Symfony Developmentsymfony 1.4, Symfony FrameworkKonrad Podgórski

Call to undefined method myUser::setReferer.

this is a common mistake after installing sfDoctrineGuardPlugin

open following class

apps/frontend/lib/myUser.class.php

and change extending class to sfGuardSecurityUser instead of default sfBasicSecurityUser

now code should look like this

< ?php

class myUser extends sfGuardSecurityUser
{
   // code goes here ...
}
